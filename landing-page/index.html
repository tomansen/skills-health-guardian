Features

Everything you need to keep skills healthy

A comprehensive toolkit for monitoring, diagnosing, and fixing your AI agent skill environment — all in one place.

🔍

Comprehensive Scanning

Scans dependency conflicts, runtime issues, and configuration problems across all your skills in seconds.

📊

Health Scoring

0–100 score with clear pass/warning/fail thresholds for every skill. Know exactly what needs attention.

🛡️

Auto-Fix

One-click fix for common issues: missing dependencies, broken paths, missing env templates, and more.

📈

Trend Tracking

Monitor health changes over time with snapshot history. Catch regressions before they become problems.

How It Works

Three steps to healthier skills

From scanning to fixing — get your entire skill environment under control in minutes.

1

Scan

Run one command to scan all skills in your workspace.

$ shg scan --all
2

Diagnose

Get a detailed report with scores and issue breakdown.

$ shg report
3

Fix

Apply suggested fixes or run auto-fix with one command.

$ shg fix

Quick Start

Up and running in 3 steps

No complex setup required. Clone, scan, and view your first health report in under a minute.

Step 1

Clone the repository

Get the source code from GitHub.

# Clone the repo git clone https://github.com/tomansen/skills-health-guardian.git cd skills-health-guardian
Step 2

Run your first scan

Scan your skills directory instantly.

# Run health check bash scripts/health-check.sh # Or use Python scanner python scripts/scanner.py
Step 3

View the report

Open the generated HTML report.

# Open report in browser open reports/skills-health.html # macOS xdg-open reports/skills-health.html # Linux
