## 🧑‍💻 About Me

<!-- 自我介绍：展示个性 + 技术热情 -->

I'm an **AI Agent developer** who believes in building things that actually work — not just demos.

My journey started with WeChat Mini Programs and cloud development, then evolved into the fascinating world of **MCP (Model Context Protocol)** and autonomous agent systems. Now I spend most of my time:

- 🔧 Building open-source tools for the **AI Agent ecosystem** — making agents more observable, healthier, and easier to manage
- 🏗️ Designing **agent team workflows** that mirror real engineering practices (because AI should ship like real software)
- 🌍 Bridging **Chinese developer communities** with global open-source standards

I'm a firm believer in: *Ship it. Measure it. Improve it.* （做出来，跑起来，迭代好。）

**Fun fact**: I manage a digital team of 9 AI "employees" (数字员工) to help run my projects. Yes, really.

## ⭐ Featured Projects

### 🛡️ [Skills Health Guardian (SHG)](https://github.com/tomansen/skills-health-guardian)
*Skills 环境健康管家 — 让你的 Agent Skills 永远保持最佳状态*

<p>
  <img src="https://img.shields.io/badge/status-active-brightgreen?style=flat-square" alt="status" />
  <img src="https://img.shields.io/badge/language-Python-3776AB?style=flat-square&logo=python&logoColor=white" alt="Python" />
  <img src="https://img.shields.io/badge/license-MIT-blue?style=flat-square" alt="license" />
</p>

A comprehensive health monitoring & management platform for AI Agent Skills:
- 🔍 **Dependency Scanning** — 自动发现并分析所有 Skill 依赖项
- ⚠️ **Conflict Detection** — 版本冲突与兼容性检测
- 📊 **Health Scoring** — 0-100 健康评分仪表盘
- 🩹 **Auto-Fix Suggestions** — 一键修复建议与执行
- 📰 **Multi-format Reports** — Markdown / JSON / HTML 可视化报告
